# HTML Validation Project

Этот проект использует W3C HTML валидатор для проверки сайта на соответствие стандарту HTML.

## Описание

Данный проект включает сайт, на котором проводится тестирование HTML-кода с помощью валидатора W3C.

## Используемые технологии

- HTML
- GitHub Actions
- W3C HTML Validator
